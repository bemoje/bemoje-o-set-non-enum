oSetNonEnum

Adds a non-enumerable property on an object

oSetNonEnum(o: object, key: string, value: string, writable: boolean, configurable: boolean): void
Parameters
o (object) The object on which to set a property value
key (string) The property key name
value (string) The property value
writable (boolean = true) Whether or not the property should be writable.
configurable (boolean = false) whether or not the property should be configurable
Returns
void:
